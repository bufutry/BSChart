BSChart is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BSChart"
```

## Also available loading charts from JSON file (see Demo)

## Requirements
  * iOS 10.0 or higher
  * ARC
